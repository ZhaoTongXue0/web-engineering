# markdownLint-config

> 文档书写规范

## 安装

依赖于 [markdownlint](https://www.npmjs.com/package/markdownlint)

```bash
npm install markdownlint-config markdownlint --save-dev
```

## 使用

在 `.markdownlint.json` 中继承本包:

```json
{
  "extends": "markdownlint-config-config"
}
```
